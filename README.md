# 🚀 AI Career & Course Recommendation System

## 📌 Overview

This project is a **rule-based AI system** developed using **Python (Tkinter GUI)** that recommends suitable **career paths and courses** based on user inputs such as **interests and skills**.

It simulates how an **expert system** makes decisions using **knowledge representation and inference techniques** from Artificial Intelligence.

---

## 🎯 Features

* 🎨 Modern dark-themed GUI using Tkinter
* 🧠 Rule-based AI decision system
* 📚 Career + Course recommendations
* 🌱 Includes real-world domains like **Agriculture, Environment, Finance, etc.**
* 🔄 Dynamic user input (Interest + Skills)
* 💡 Covers **20+ interest areas**
